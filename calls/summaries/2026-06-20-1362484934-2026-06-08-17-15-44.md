## Клиент

Собеседник — основатель стартапа, который занимается автоматизацией (платформа с workflow-моделью, нодами, агентом-скаутом, автозвонками через ИИ). Клиенты стартапа — селлеры маркетплейсов. Есть ~30 партнёров, реферальная программа до 45% от выручки. Платформа многоязычная. Команда полностью на аутсорсе.

## Ключевые факты

- Клиент ищет решение для внутренней автоматизации: быстрый онбординг новых сотрудников, снижение нагрузки на основателя от ответов на типовые вопросы.
- У клиента уже есть нейронка, которая записывает все разговоры (все транскрипты хранятся). Также есть аккаунт OpenAI на базе которого настроен ассистент, но токены быстро сгорают.
- Клиент хочет графовую базу знаний (RAG-агент с графовой структурой, GRAG) для хранения и поиска информации.
- Цель клиента: агент контролирует отделы (маркетинг, продажи), обучает сотрудников через базу знаний, квалифицирует лиды (10 000 исходящих сообщений в день).
- У клиента работает голосовой агент для обзвона лидов — протестирован, за первые часы дал 41 лид при расходе пакета на 3114 контактов.

## Решения

- MarketAI24 (команда 3-4 человека) предлагает: оцифровать и структурировать базу знаний клиента, построить RAG-агент с графовой БД, реализовать интерфейс в Telegram-боте.
- Использовать OpenAI API (по тарификации за токены), не локальный деплой.
- Вилка стоимости: 30–100 тысяч рублей. Срок: 2–3 недели.
- Схема работы: 50% предоплата, договор с ИП, 50% по завершению с правками.
