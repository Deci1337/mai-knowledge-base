## Ключевые факты и инсайты
- Обсуждение редизайна лендинга, AI-ассистента и ускорения дашборда.
- Упрощение интерфейса: убираются видео на заднем фоне, оставляется только номер регистрации.
- Добавление новых блоков для аналитики и финансов для селлеров.
- Введение UTM-ссылок для отслеживания переходов.
- Обсуждение необходимости быстрой загрузки данных на дашборде.

## Решения
- Убрать имя пользователя из формы, оставить только номер регистрации.
- Добавить блоки с новыми UTM-ссылками и аналитикой.
- Ввести два оффера: кредит для селлеров и услуги по аналитике.
- Вернуть страницу AI-провайдера в админке для доступа к нейронным сетям.
- Добавить кнопки с предпромптами для AI-ассистента, чтобы пользователи могли быстро получать отчеты.

## Изменения в процессах
- Упрощение формы для пользователей: убираются лишние поля, добавляются только необходимые.
- Введение системы заявок на подключение AI-ассистента с автоматическим подтягиванием номера телефона.
- Создание кэширования данных для ускорения загрузки дашборда, с фокусом на последние две недели.
- Разработка системы, где пользователи могут выбирать готовые промпты для получения отчетов.

## Прочие договорённости
- Необходимо, чтобы AI-ассистент мог работать с данными из базы, но без сложной интеграции на первом этапе.
- Важно, чтобы пользователи могли легко оставлять заявки на подключение и получать уведомления о новых предложениях.
- Обсуждение необходимости визуального оформления и простоты интерфейса для пользователей.
