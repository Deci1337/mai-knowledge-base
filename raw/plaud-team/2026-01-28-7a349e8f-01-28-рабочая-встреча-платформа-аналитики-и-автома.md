## 01-28 Рабочая встреча: платформа аналитики и автоматизации для маркетплейсов (WB/Ozon) — API-мэппинг и Telegram-бот

**Дата:** 2026-01-28  
**Тип:** Внутренняя встреча

### Участники

- Roman Vassiliev
- Speaker 3
- Speaker 1

```markdown
## Ключевые факты и инсайты
- Разработана API для отображения динамики кабинета, включая финансовые отчеты, план-факт, маржу, заказы и продажи.
- Внедрена система трекинга задач для менеджеров через Telegram-бота, который проверяет выполнение задач и их влияние на ключевые метрики (например, CTR).
- Создана AI-система для предоставления рекомендаций на основе данных, с возможностью интеграции с нейронными сетями.
- Партнерская программа: 45% от выручки отдается партнерам за привлечение клиентов, сервис стоит 5000 рублей за кабинет.
- Ведется работа над интеграцией с банками и другими сервисами, такими как Ростелеком.

## Решения
- Определено, что для успешной работы с API необходим специалист, который разбирается в маркетплейсах и может сопоставить данные с API WB и OZON.
- Установлено, что задачи должны быть четко прописаны с дедлайнами и контрольными точками для отслеживания выполнения.

## Изменения в процессах
- Внедрение конструктора правил для автоматизации задач менеджеров на основе метрик.
- Создание бота для анбординга, который будет обучать пользователей и отслеживать их прогресс.
- Установление системы отчетности, где данные будут проверяться и автоматизироваться для упрощения работы с таблицами.

## Прочие договорённости
- Все задачи должны быть четко прописаны и согласованы с ответственными лицами.
- Необходимо подписывать НДА для защиты информации, связанной с платформой.
- Связь с программистами и другими участниками проекта должна быть организована для эффективного выполнения задач.
```
