## 01-15 Рабочая встреча: координация операционного директора, финансовые сверки RNP, стабильность API Wildberries и онбординг селлеров

**Дата:** 2026-01-15

- Roman Vassiliev
- Speaker 1
- Speaker 4

```markdown
## Ключевые факты и инсайты
- Обсуждение работы с API Wildberries и его ограничениями. В частности, данные по рекламным кампаниям иногда возвращаются пустыми, что связано с обновлениями рекламного кабинета.
- Участники встречи отметили, что данные по зонам в API могут быть более полными, чем в самой системе Wildberries.
- Обсуждение необходимости сверки финансовых отчетов и данных, получаемых через API, для выявления расхождений. Обсуждались конкретные цифры по продажам и перечислениям за товар.

## Решения
- Определено, что для сверки финансовых отчетов нужен человек, который будет работать с программистом для проверки корректности данных.
- Установлено, что необходимо проводить ежедневные и еженедельные сверки данных, чтобы обеспечить точность отчетности.
- Принято решение о проведении консультаций по анализу данных и стратегии работы с клиентами.

## Изменения в процессах
- Внедрение нового подхода к работе с данными: создание универсальной таблицы, где пользователи смогут выбирать необходимые метрики и блоки информации.
- Обсуждение необходимости онбординга для нового сервиса, чтобы пользователи могли легко ориентироваться в системе.

## Прочие договорённости
- Установлено время для регулярных созвонов: 8:00 по Москве (10:00 по Екатеринбургу) для обсуждения текущих задач и сверки данных.
- Договоренность о том, что Лёня будет заниматься автоматизацией через API и проверкой данных, которые не поступают из API.
- Участники согласились на проведение микроконсультаций по анализу данных, где первая консультация будет бесплатной, а дальнейшая работа будет оцениваться.
```
