## Вопросы клиента
- **Можно ли задать вопросы о бизнесе?**  
  Ответ: Да, вопросы нужны для понимания, чем MarketAI24 может быть полезен.

- **Как работает внутренняя реклама?**  
  Ответ: Внутренняя реклама анализируется по метрикам CTR, CPM, CPC с разбивкой по поиску и полкам. ИИ помогает выявлять проблемы и ставить задачи менеджерам для оптимизации.

- **Как ИИ помогает с рекламой и задачами?**  
  Ответ: ИИ (Иишка) автоматически ставит задачи менеджерам по рекламе, отгрузкам, логистике, контролирует выполнение и оповещает при отклонениях от норм.

- **Как сравнивать свои показатели с конкурентами?**  
  Ответ: Через сервис Эвирма и Валбис можно получить данные по конверсиям и запросам конкурентов, которые интегрируются в платформу для анализа и постановки задач.

- **Как работает SEO-продвижение карточек?**  
  Ответ: SEO влияет на конверсию, и для улучшения карточек можно заказать услуги фрилансеров или использовать внутренние инструменты для сбора семантического ядра.

- **Можно ли посмотреть кабинет и данные на телефоне?**  
  Ответ: Нет, для полноценного просмотра и демонстрации нужен компьютер с экраном, камера и микрофон не обязательны.

- **Как получить доступ к платформе, если не приходит SMS с кодом?**  
  Ответ: Проверить номер телефона, использовать правильную ссылку для регистрации, отключить VPN. Если проблемы сохраняются, нужна дополнительная техническая поддержка.
