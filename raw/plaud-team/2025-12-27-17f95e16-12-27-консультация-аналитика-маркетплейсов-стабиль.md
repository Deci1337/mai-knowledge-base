## 12-27 Консультация: аналитика маркетплейсов, стабильность выгрузок и интеграции API (Wildberries/Ozon, финотчеты, DataLens)

**Дата:** 2025-12-27  
**Тип:** Внутренняя встреча

### Участники

- Роман
- Speaker 2

```markdown
## Ключевые факты и инсайты
- В команде MarketAI24 обсуждаются вопросы аналитики маркетплейсов, стабильности выгрузок и интеграции API для Wildberries и Ozon.
- В отчете по артикульной аналитике за 2025 год содержится 125 колонок с данными, включая штрафы, доплаты и рекламную статистику.
- Проблемы с выгрузками данных из Wildberries и Ozon: часть данных недоступна без премиум аккаунтов.
- В текущий момент команда тестирует несколько кабинетов для получения данных и сверки с дашбордами.
- Используется отчет «Рука на пульсе» для ежедневного мониторинга расходов на рекламу и заказы.
- В команде есть необходимость в аналитике и мониторинге изменений API.

## Решения
- Программисту будет предоставлен доступ к необходимым данным и токенам для работы.
- Запланирован созвон с программистом для уточнения формата данных и необходимых запросов.
- Ожидается, что работа будет выполнена в течение двух дней, с возможностью связи в выходные.

## Изменения в процессах
- Внедрение автоматизации для проверки выполнения задач менеджерами, с использованием робота для анализа результатов.
- Переход на новый сервис для работы с данными, так как текущий сервер не оплачивается.

## Прочие договорённости
- Участники созвона согласовали, что программист свяжется с аналитиком для уточнения деталей работы.
- Ожидается, что аналитик предоставит информацию о своих услугах и опыте работы.
- Обсуждение возможности коллаборации и реферальной программы для привлечения новых клиентов.
```
