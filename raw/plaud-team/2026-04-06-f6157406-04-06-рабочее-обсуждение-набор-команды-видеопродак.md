## 04-06 Рабочее обсуждение: набор команды видеопродакшна и стратегия обучения с высоким порогом входа

**Дата:** 2026-04-06  
**Тип:** Внутренняя встреча

### Участники

- Speaker 1
- Speaker 2

```markdown
## Ключевые факты и инсайты
- Гриша создает видеоролики, которые имеют высокий уровень вовлеченности.
- В команде видеопродакшна планируется набор людей, готовых к обучению, с акцентом на самоотдачу.
- Из 98 видов контента 45% предназначены для обучения, 30% — для клиентов, 25% — для партнерских программ.
- Для отбора участников обучения будет установлен взнос в 50 тысяч рублей, чтобы отсеять менее мотивированных людей.

## Решения
- Набор в команду будет осуществляться только среди тех, кто готов жертвовать временем и ресурсами для обучения.
- Будет создан отдельный аккаунт для обучения, чтобы не терять охват на основном аккаунте, где размещаются успешные ролики.

## Изменения в процессах
- Минимальный размер группы для обучения будет составлять 20 человек, чтобы участники могли учиться друг у друга и задавать вопросы.
- Внутренние обсуждения будут записываться и анализироваться для улучшения коммуникации и планирования.

## Прочие договорённости
- Первым коммерческим клиентом станет Роман.
- Будет проведена распаковка, чтобы создать видеоролик, который привлечет лиды.
- Все разговоры будут документироваться для дальнейшего анализа и улучшения процессов.
```
