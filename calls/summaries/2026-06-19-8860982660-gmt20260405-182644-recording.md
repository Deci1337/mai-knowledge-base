## Ключевые решения и факты
- Реклама: два типа выгрузки — Fullstat (общая по всем кампаниям) + кластерная статистика по каждой кампании
- Формула: общая статистика минус статистика поискового кластера = статистика рекомендательной полки; проверено на 10x-референсе
- Остатки: разбирали учёт по складам, включая транзит и FBO
- Расчёт плана возвратов: % выкупа применяется к заказам
- Оборачиваемость: формула дней запасов для каждого артикула
- Финансовый отчёт: сверка данных ВБ с внутренними расчётами
- Отзывы: обсудили процесс отслеживания и ответа
- Инструменты: GPT/Codex для автоматизации Python-скриптов; арендован сервер для запуска скриптов
- Платформа market.ai24 медленно загружается из-за большого числа подключённых кабинетов
