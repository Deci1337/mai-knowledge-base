## 01-12 Совещание: оценка качества бота, стратегия MyGPT/Market AI, монетизация и партнёрства

**Дата:** 2026-01-12  
**Тип:** Внутренняя встреча

```markdown
## Ключевые факты и инсайты
- Обратная связь от менеджмента о текущем состоянии бота: низкое качество ответов, не соответствующие ожиданиям.
- Критерии оценки работы бота: 
  - 0 баллов — неверные данные.
  - 1 балл — одна некритичная ошибка.
  - 2 балла — несколько ошибок.
  - 3 балла — правильные ответы.
- Проведен частотный анализ реальных запросов, результаты загружены в Excel для оценки менеджерами.
- Необходимость создания бота, который будет выступать в роли спарринг-партнера для локальных команд, чтобы улучшить качество принятия решений.
- Важность наличия системы оценки для достижения нормального уровня удержания клиентов (ретеншн).
- Проблемы с базовыми метриками: скорость доставки, реклама, SEO, рейтинг. Неправильные данные могут привести к снижению конверсии.
- Успешные примеры из других компаний: Axel Club, который выкупает магазины на Amazon и управляет ими через экономию на масштабе.

## Решения
- Создание системы, которая будет обучать бота на основе реальных данных и запросов.
- Разработка алгоритма с регламентами и обучающими материалами для улучшения работы бота.
- Стратегия привлечения инвестиций: 30 миллионов для роста пользователей, 70 миллионов для покупки микросервисов.
- Упрощение процесса для менеджеров, чтобы снизить точку входа к продуктам.

## Изменения в процессах
- Внедрение системы оценки работы бота для улучшения качества ответов.
- Создание отдельного продукта MyGPT для привлечения пользователей и улучшения охвата.
- Разработка стратегии по привлечению клиентов через рекомендации и реферальные программы.

## Прочие договорённости
- Необходимость создания базы данных для поддержки пользователей и улучшения качества обслуживания.
- Участие в тендерах для кредитования селлеров, что позволит расширить клиентскую базу.
- Согласование с банками условий для получения кредитов на развитие бизнеса и недвижимости.
```
