## Вопросы клиента  
- **Как работает подсветка пробелов и аномалий?**  
  Ответ: Система собирает все данные, чтобы нейронная сеть могла искать взаимосвязи и аномалии. Автоматически ставятся задачи менеджерам на основе бизнес-процессов и метрик. Если задача выполнена и влияет положительно, она закрывается, если нет — возвращается в работу.  
- **Можно ли выгружать данные в Excel?**  
  Ответ: Да, для удобства «староверов» предусмотрена выгрузка в Excel с возможностью настройки под нужды.  
- **Как настраиваются связки товаров и менеджеров?**  
  Ответ: Можно создавать связки, давать им имена, сохранять, редактировать и удалять. Это помогает управлять товарами, которые связаны между собой (например, лид-магнит и сопутствующие товары).  
- **Подтягиваются ли теги из Wildberries?**  
  Ответ: Нет, теги подтягиваются разово при настройке, не обновляются автоматически.  
- **Как рассчитываются остатки и продажи по регионам и складам?**  
  Ответ: Остатки берутся на определенную дату (обычно на 00:00 выбранного дня). Продажи учитываются по группе складов (региону). Есть возможность исключать дни без остатков при расчете средних продаж.  
- **Как работает модуль планирования допоставок?**  
  Ответ: Система рассчитывает необходимое количество товара для отгрузки с учетом средних продаж, запаса дней, кратности упаковки и ограничений поставщиков.  
- **Как работает ИИ-помощник для менеджеров?**  
  Ответ: ИИ работает на базе заранее настроенных правил (нод), подсказывает действия при отсутствии идей у менеджера. Настройка стоит 200 тыс. руб. разово, подписка 2 тыс. руб. в месяц. Без настройки ИИ может выдавать некорректные рекомендации.  
- **Какие условия тестового периода и оплаты?**  
  Ответ: Тестовый период 2 дня, после чего можно оплатить 5 тыс. руб. за месяц использования одного кабинета. Есть возможность расширения по сотрудникам, но пока не реализовано.  
- **Как происходит интеграция с маркетплейсами и API-ключами?**  
  Ответ: Для полного функционала нужно подключить авторизованный сервис с платным токеном. Клиент получает доступ только на чтение.  
- **Как рассчитывается план-факт по продажам и марже?**  
  Ответ: Можно настраивать план по разным метрикам (маржа, заказы, продажи) с учетом сезонности и распределения по дням. Есть фильтры по категориям, менеджерам, связкам.  
- **Как учитываются выкупы и внешняя реклама?**  
  Ответ: Менеджеры указывают самовыкупы и расходы на внешнюю рекламу, которые вычитаются из заказов для корректного анализа.  
- **Как рассчитывается кредитоспособность для банков?**  
  Ответ: Система передает обезличенные финансовые данные (продажи, перечисления, оборачиваемость остатков) банкам для оценки риска дефолта и выдачи кредитов селлерам.
