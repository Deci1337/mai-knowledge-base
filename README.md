# База знаний

Автоматически пополняется агентом 1 (MarketAI).

## Структура

| Папка | Содержимое |
|---|---|
| `hub-growth/` | Маркетинг, контент, трафик, партнёры, лидогенерация |
| `hub-sales/` | Продажи, диагностики, скрипты, закрытие сделок |
| `hub-product/` | Продукт, разработка, AI, аналитика, техподдержка |
| `hub-finance/` | Финансы, операции, бухгалтерия, инвестиции |
| `zoom-team/` | Командные созвоны: выжимка + все факты и решения |
| `zoom-leads/` | Клиентские созвоны: анализ менеджера + выжимка |
| `client-feedback/` | Жалобы и пожелания клиентов из созвонов |
| `telegram-digest/` | Ежедневные дайджесты из рабочих Telegram-чатов |
| `general/` | Общие материалы о компании |

## Формат документов

```yaml
---
title: Название
type: team | lead | internal | client | client-feedback
source: zoom | telegram-digest | manual
date: YYYY-MM-DD
---
```

Перекрёстные ссылки: `[[название-файла]]`

## Источники

- **Zoom** — транскрипты каждые 6 часов автоматически
- **Telegram** — дайджест ежедневно в 21:00
- **Ручная загрузка** — `/upload` юзерботу агента 1
