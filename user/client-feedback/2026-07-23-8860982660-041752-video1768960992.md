> Жалобы и пожелания клиента из звонка.

**Сотрудник:** Амир Хайруллин  
**Дата:** 23.07.2026 04:17

## Жалобы и боли
- Менеджер видит много данных в отчетах и постоянно забывает совершить какое-то действие.
- Менеджмент хромает, менеджеры что-то забывают сделать.
- Бывали ситуации, когда нужны были деньги на развитие бизнеса.
- В офисе стало больше работы по контролю за остатками и отгрузками.
- Данные из воронки продаж могут отличаться от данных в финансовом отчёте.
- Не всегда разумные рекомендации от системы по распределению товара по складам.
- Необходимость постоянно отслеживать и контролировать остатки и заказы.
- Есть проблема с выбором товара для масштабирования бизнеса.
- Загрузка данных может занимать до суток, что влияет на оперативность работы.

## Пожелания
- Хотелось бы более подробно познакомиться с сервисом и понять, чем он может быть полезен.
- Возможность выгружать отчёты в Excel для удобства просмотра.
- Интеграция с нейронными сетями для получения рекомендаций на базе данных.
- Возможность фильтровать данные по связке, менеджеру, артикулу.
- Настройка задачника (CRM) для менеджеров с подсказками по складам и заказам.
- Возможность выстраивать локализацию и выбирать склады для отгрузок.
- Подключение партнерских сервисов для получения лучших условий по доставке и внешнему трафику.
- Внедрение AI-ассистента для автоматизации и помощи в работе.
- Предоставление тестового периода на рабочую неделю для погружения в сервис.
- Проведение дополнительного созвона для обсуждения вопросов и обратной связи по системе.
