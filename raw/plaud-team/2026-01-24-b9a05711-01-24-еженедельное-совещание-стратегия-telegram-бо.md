## 01-24 Еженедельное совещание: Стратегия Telegram-бота и привлечение пользователей

**Дата:** 2026-01-24  
**Тип:** Внутренняя встреча

### Участники

- Roman Vassiliev
- Speaker 2
- Speaker 3

```markdown
## Ключевые факты и инсайты
- Обсуждение точности ответов нейронных сетей и людей. Документация считается более надежной.
- Роман Вассильев выразил опасения по поводу выполнения плана по количеству пользователей в январе из-за незавершенности продукта.
- Максим и Андрей подготовили трафик для привлечения пользователей.
- Стратегия включает продажу бета-продукта и удержание пользователей через Telegram-бота с продуманным стори-лайном.

## Решения
- Создание Telegram-бота, который будет использоваться для привлечения трафика и удержания пользователей.
- Бот будет иметь структуру: Guest, Trial, Paid, Beta, Expired.
- Пользователи будут получать сообщения, которые будут прогревать их интерес и подводить к покупке.
- Внедрение возможности для пользователей корректировать свой стори-лайн через Excel или другие инструменты.

## Изменения в процессах
- Необходимость создания документации для бота, чтобы нейронка использовала только проверенную информацию.
- Стандартный стори-лайн будет разработан для пользователей, с возможностью его редактирования.
- Внедрение автоматизированных инструкций в бота для упрощения процесса создания UTM-меток и реферальных ссылок.

## Прочие договорённости
- Все участники должны работать над формированием технического задания (ТЗ) для бота.
- Необходимо создать цепочку сообщений для бота, которая будет включать утренние, дневные и вечерние сообщения.
- Участники согласились, что бот должен быть ориентирован на продажу и удержание пользователей, а не на предоставление лишней информации.
```
