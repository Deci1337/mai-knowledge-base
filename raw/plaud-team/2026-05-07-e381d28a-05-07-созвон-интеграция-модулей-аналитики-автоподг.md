## 05-07 Созвон: интеграция модулей аналитики, автоподгрузка данных и партнёрская программа

```markdown
## Ключевые факты и инсайты
- Моталка закрывается при окончании пробного периода, если не списались средства. Необходимо улучшить отображение статуса пробного периода.
- В текущей версии системы отсутствует возможность отправки изображений в поддержку, что вызывает проблемы с коммуникацией.
- Внедрение демо-доступа и рекуррентных платежей завершено.
- Партнёрская программа требует доработки, необходимо исправить отображение для зарегистрированных пользователей.
- В процессе интеграции с поставщиками необходимо обеспечить автоматическую подгрузку данных с API, так как текущая система работает в ручном режиме.
- Используемая база данных: PostgreSQL. Рассматривается возможность интеграции с Clickhouse для повышения скорости работы.
- У конкурентов есть модули, которых нет у нас, такие как АБЦ-анализ, география продаж и история остатков.

## Решения
- Необходимо создать отдельные партнёрские модули для рынка, с возможностью автоматического начисления комиссий.
- Для автоматизации подгрузки данных с API требуется доработка текущей системы.
- Запланирован созвон с разработчиками для обсуждения архитектуры и возможности интеграции с их системой.

## Изменения в процессах
- Уведомления о новых заявках и подключениях должны быть реализованы в админ-панели.
- Внедрение автоматического сопоставления данных из разных отчетов для упрощения проверки и выявления ошибок.
- Создание таблиц для сравнения данных из разных источников, чтобы упростить процесс проверки.

## Прочие договорённости
- Назначен созвон с разработчиками для обсуждения интеграции и архитектуры системы.
- Необходимо подготовить таблицы для сверки данных и отправить их команде.
- Обсуждение возможности курирования проекта одним из членов команды для повышения эффективности работы.
```
