## Ключевые факты и инсайты
- В среднем пользователи делают 5 запросов за сессию, однако некоторые делают только 2. Это вызывает вопросы о качестве продукта и конверсии.
- Логирование действий пользователей необходимо для анализа поведения и выявления причин отвалов.
- Персональные данные пользователей должны быть обезличены для соблюдения законодательства.
- Необходимо определить, сколько пользователей зарегистрируется и сколько из них будет активно задавать вопросы.
- Основные конкуренты MyGPT: Синтекс, AI, ЧАД, GPT, ЧСД, ПАБА, МАША GPT.
- Для достижения окупаемости необходимо привлечь 100 пользователей в первый месяц.

## Решения
- Запуск логирования действий пользователей для анализа их поведения и выявления проблем.
- Регистрация пользователей через упрощенный интерфейс, где они могут начать взаимодействие с продуктом без оплаты.
- Использование блогеров для продвижения продукта, с акцентом на широкий сегмент аудитории, включая тех, кто генерирует контент с помощью нейросетей.
- Определение бюджета на маркетинг: 2 миллиона рублей на различные каналы (таргет, блогеры, MLM и т.д.).

## Изменения в процессах
- Упрощение процесса регистрации на MyGPT: пользователи смогут зарегистрироваться одной кнопкой через Telegram.
- Необходимо провести сплит-тестирование различных версий лендинга для определения наиболее эффективной.
- Установление четких KPI для новых сотрудников, чтобы оценить их эффективность в течение испытательного срока.

## Прочие договорённости
