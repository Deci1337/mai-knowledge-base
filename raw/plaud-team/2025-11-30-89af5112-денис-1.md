```markdown
## Ключевые факты и инсайты
- У команды есть проблемы с недостатком сна, что влияет на продуктивность.
- В команде обсуждается необходимость создания сервера для работы с доменом и API.
- Обсуждение о необходимости подключения платежной системы для получения денежных средств.

## Решения
- Создание сервера на облачной платформе с использованием Ubuntu, выбор Нидерландов для возможности использования VPN и обработки запросов к ChatGPT.
- Принято решение о том, что конечный продукт должен позволять пользователям подписывать свою API и получать всю необходимую информацию.
- Для получения инвестиций будет создана форма заявки, которая будет отправлять номер телефона и ID кабинета пользователя.

## Изменения в процессах
- Необходимо доработать систему управления пользователями, так как текущая система входа не работает.
- В дашборде необходимо добавить себестоимость хранения, которая будет подтягиваться через API.
- Внедрение планирования по месяцам для каждого артикула, с возможностью автоматического подсчета плана на год на основе месячных планов.

## Прочие договорённости
- Важно перепроверить все цифры в дашборде перед презентацией.
- Необходимо настроить уведомления о состоянии сервера через Telegram-бота.
- Домен будет подключен к серверу, и дальнейшие изменения в домене будут возможны без значительных затрат.
```
