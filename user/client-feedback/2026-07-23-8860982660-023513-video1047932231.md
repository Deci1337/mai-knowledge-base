> Жалобы и пожелания клиента из звонка.

**Сотрудник:** Амир Хайруллин  
**Дата:** 23.07.2026 02:35

## Жалобы и боли
- Плохое качество связи по телефону, из-за чего сложно слышать собеседников.
- Отсутствие нормального решения для ежедневного контроля и анализа данных по маркетплейсам.
- Ошибки и недоработки в текущей версии сервиса (бета-версия), например, не учитываются возвраты по сумме к перечислению.
- Задержки в загрузке данных и необходимость доработок для стабильной работы.
- Неудобство в отображении отчетов по неделям, особенно если месяц начинается не с понедельника.
- Отсутствие пока полностью налаженной технической поддержки и команды для взаимодействия.
- Возможные «фантазии» искусственного интеллекта, если менеджер не думает, что может привести к излишним рекомендациям.
- Необходимость интеграции с внутренними регламентами компании для корректной работы ИИ.

## Пожелания
- Возможность видеть юнит-экономику и рекламный кабинет в одном месте с ежедневным обновлением данных.
- Автоматическое формирование задач на основе метрик и бизнес-процессов с проверкой их выполнения роботом.
- Гибкая настройка планов продаж и маржи по артикульному уровню с учетом сезонности.
- Возможность сравнивать периоды и видеть динамику изменений.
- Интеграция с банками-партнерами для получения льготного финансирования на основе обезличенных данных.
- Возможность выгрузки отчетов в Excel для удобной работы с метриками.
- Настраиваемая управленческая отчетность с возможностью просмотра нескольких кабинетов и детализацией по артикулам.
- Внедрение тепловой карты и автоматического расчета поставок с уведомлениями.
- Распределение доступа сотрудников к данным и карточкам товаров.
- Подключение нейросетей для автоматизации и помощи в работе с контентом и аналитикой.
- Партнерская программа с возможностью заработка до 45% от выручки.
- Бесплатное пользование сервисом при условии рекомендации другим пользователям.
- Возможность интеграции API с ограничением доступа только на чтение для безопасности.
